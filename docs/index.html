Notes

La Banque centrale européenne publie les taux de change de référence de l’euro (EUR), généralement mis à jour vers 16h00 CET les jours ouvrés.

L’accès aux données s’effectue via le ECB Data Portal, en s’appuyant sur le standard SDMX (Statistical Data and Metadata eXchange).

ECB Data portal API Overview

Structure générale d’une clé EXR

Une clé EXR permet d’identifier de manière unique chaque série de taux publié.

Elle est composée de cinq dimensions. Chaque segment décrit respectivement la fréquence des données, la devise cotée, la devise de référence, le type de taux de change et la nature de la série.

FREQ.CURRENCY.CURRENCY_DENOM.EXR_TYPE.EXR_SUFFIX.
